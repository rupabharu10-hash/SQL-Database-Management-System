# SQL-Database-Management-System
This academic project focuses on designing and implementing a relational database system using SQL. The objective of the project is to understand core database concepts such as table creation, data manipulation, normalization, and query optimization by working with structured datasets.
# Amazon Sales Data Analysis using SQL

## Project Overview
This project analyzes Amazon sales data to gain insights into sales performance
and customer behavior across different branches using SQL.

## Dataset Description
- 1000 sales records
- 3 branches: Mandalay, Yangon, Naypyitaw
- 17 columns including sales, customer, product, and rating details

## Objectives
- Analyze branch-wise and city-wise sales performance
- Identify top-selling product lines
- Understand customer behavior and preferences
- Evaluate revenue, VAT, and gross income
- Perform payment method and rating analysis

## SQL Concepts Used
- DDL & DML operations
- Aggregate functions (SUM, AVG, COUNT, MAX, MIN)
- GROUP BY and HAVING
- Subqueries
- Date-based analysis

## Technologies Used
- SQL
- MySQL / PostgreSQL / Oracle

## Key Insights
- Identified highest revenue-generating branches
- Determined top-performing product categories
- Analyzed customer purchasing patterns
- Evaluated profitability using gross income
